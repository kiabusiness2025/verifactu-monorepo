# VeriFactu Brand Assets

Estructura organizada de assets de marca y branding.

## Estructura de Carpetas

### /favicon
Archivos de favicon para navegadores y dispositivos:
- favicon-16.png (16x16)
- favicon-32.png (32x32)
- favicon-48.png (48x48)
- favicon.ico (multi-size)
- apple-touch-icon.png (180x180 para iOS)

### /logo
Logos principales de VeriFactu en diferentes variantes:
- verifactu-logo.png (logo principal)
- verifactu-logo-light.png (para fondos claros)
- verifactu-logo-dark.png (para fondos oscuros)
- verifactu-logo-variant.png (variante alternativa)
- verifactu-official.jpg (logo oficial)

### /icon
Iconos de aplicación y otros assets:
- app-icon-512-v2.png (512x512 - versión 2)
- app-icon-512-v3.png (512x512 - versión 3)

### /social
Assets para redes sociales (OG images, banners, etc.)
Actualmente vacío - agregar según necesidad.

## Uso

### En Landing (apps/landing)
Los favicons se referencian desde /public/brand/favicon/

### En App (apps/app)
Los iconos de app se pueden importar desde este directorio.

## Notas
- Mantener nombres consistentes y descriptivos
- Usar PNG para transparencias
- JPG solo para fotos/imágenes complejas
- SVG preferido cuando sea posible (escalable)
